Listes :

<ol> </ol> Ordered List (Liste ordonnée)

<ul> </ul> Unordered List (Liste NON ordonnée)

<li> </li> Pour les éléments de listes.

Peuvent s'imbriquer sans soucis. (Penser à TAB)

list-style-type:none; (Pour supprimer les puces)

Tableaux :

<table> </table> Tout début et tout à la fin du tableau pour délimiter

<tr> </tr> Ligne

<td> </td> Cellule

<th> </th> Cellule d'en tête

Balises Génériques

<span> Balise de type INLINE

<div> Balise de type BLOCK
